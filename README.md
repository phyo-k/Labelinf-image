# Labeling-image
--------------------------------------------------------------------------------
인공지능 - 음식 사진 분류 처리 연습 프로젝트입니다.

- 265개의 음식 이미지 사진 데이터가 있습니다.
- "국, 밥, 김치, 고기" 네 가지에 대해 분류하는 모델입니다.
- labeling tool을 사용하며, 라벨링된 값은 xml파일로 저장됩니다.
- 파일이 다 올라가지 않아 구글 드라이브 주소를 올립니다.
